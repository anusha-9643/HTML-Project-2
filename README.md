# Personal Portfolio Card

This is a simple **frontend developer portfolio card** built using HTML and CSS. It showcases a personal profile, including a photo, name, short bio, and social media links. The design is modern, clean, and responsive for small screens.

## Features

- Profile image displayed as a horizontal oval.
- Name displayed prominently under the photo.
- Short bio describing skills and interests.
- Social media buttons: GitHub, LinkedIn, Twitter.
- Hover effects on buttons for better user interaction.
- Card layout with rounded corners and shadow effect.

## Technologies Used

- HTML5
- CSS3
- Responsive design principles
